Why These Tools

React and Next.js

Component based architecture maps directly to how I think about UIs. Next.js adds the file based routing, server rendering, and deployment optimization that takes a React project to production grade fast.

Node.js and REST APIs

JavaScript end to end means one mental model, one language, faster development. REST APIs are the backbone of every product I build. I design them to be predictable, versioned, and easy to extend.

Claude Code and Cursor

I don't use AI to avoid thinking. I use it to eliminate the mechanical parts of coding so my thinking stays on architecture and product decisions. The result is noticeably faster timelines without quality loss.

Figma

Still the best tool for communicating interface decisions before they're in code. I use it for planning and client alignment but I move to code fast. Figma is a communication tool, not a deliverable.
